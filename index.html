Алгоритм для задачи о рюкзаке
Собственно вот задача, а точнее одна из популярных её вариаций. Вор пробрался в магазин техники, у него есть рюкзак объемом n единиц. В магазине, он увидел несколько вещей: Его задача оптимально уместить эти вещи в рюкзаке так, чтобы он унес драгоценностей на максимальную стоимость.
Метод динамического программирования - этот метод включает в себя работу с двумерным массивом. Способ интересный, и довольно не сложный. Он хорошо оптимизирован, потому что не включает в себя абсолютно никаких элементов перебора.
Метод жадного алгоритма - метод заключается в том, чтобы рассортировать все предметы по их удельной стоимости (то есть по цене за единицу веса), а после этого проходя по отсортированному списку искать наилучшую комбинацию.
